# 💻 Programlama Temelleri: Java Öğrenme Yolculuğu

Bu klasör, **Java programlama dilinin temel yapı taşlarından başlayarak**, algoritma kurma becerilerini geliştiren kapsamlı bir kod arşivini içermektedir.
Proje; basit sözdizimi kurallarından karmaşık veri yapılarına ve bellek yönetimine kadar **120'den fazla uygulama** ile yapılandırılmıştır.

## 🎯 Kazanımlar

Bu çalışmaları tamamlayarak şu yetkinlikler hedeflenmiştir:

* **Algoritmik Düşünme**
  Karmaşık problemleri küçük, yönetilebilir kod bloklarına ayırma.

* **Veri Yönetimi**
  Bellek dostu veri yapılarını (Arrays, Matrices) seçme ve etkin kullanma.

* **Hata Analizi**
  Syntax, Runtime ve Mantıksal hataları ayıklama (debugging).

* **Kod Okunabilirliği**
  Java isimlendirme standartlarına ve temiz kod yazımına uyum.
